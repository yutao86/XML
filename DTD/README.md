# 文档类型定义

> Document Type Definition

## Overview

- XML的文档类型定义，可以看成模板；
- 由于DTD限制较多，使用时较不方便，近来已渐被 [XML Schema](https://zh.wikipedia.org/wiki/XML_Schema) 所取代
- 组成
  - 元素（Elements）
  - 属性（Attribute）
  - 实体（Entities）
  - 注释（Comments）
