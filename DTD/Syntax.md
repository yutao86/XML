# DTD语法

> Document Type Definition Syntax

## Overview

- 元素声明语法如下：

```
<!ELEMENT 元素名称 元素內容>
```

- 属性声明语法如下：

```
<!ATTLIST 元素名称、属性注释、属性值型态、属性的內定值>
```

- 实体声明语法如下：

```
<!ENTITY 实体名称 实体內容>
```

- 注释语法如下：

```
<!-- 注释內容 -->
```
